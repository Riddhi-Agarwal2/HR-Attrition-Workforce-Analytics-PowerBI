# HR-Attrition-Workforce-Analytics-PowerBI
Project Overview
Power BI project analyzing employee attrition trends using the IBM HR dataset with DAX-based KPIs, interactive dashboard, key insights and business recommendations..

Tools Used
Power BI 
DAX (Data Analysis Expressions)
Microsoft Excel

Data preparation 
Removed irrelevant columns from the dataset
Created Age Group column for demographic analysis
Created Salary Slab column based on montly income
Verified data types and prepared data for analysis

Key KPIs developed
Total Employees
Attrition Rate
Employees Left 
Average Years
Average Age
Average Montly Salary 

Dashboard Feature
Interactive slicers and filters
Department-wise Attrition analysis
Salary and overtime impact analysis
Separate Business Insights page

Business Insights
The dashboard highlights patterns in employee turnover across departments, salary groups, and categories, helping HR teams identify areas requiring retention strategies.
